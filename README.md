# Python-Card-Game-War
A Python-based card game where two players draw cards from a shuffled deck and the highest card wins each round. Built using OOP concepts with classes for Card, Deck, Player, and Game. A great beginner-friendly project to learn Python, classes, objects, and basic game logic.

This is a simple Python card game where two players draw cards from a shuffled deck and the player with the highest card wins the round. The game continues until the deck runs out or the user quits.

## Features
- Card, Deck, Player, and Game classes
- Object-Oriented Programming
- Randomized shuffled deck
- Interactive terminal gameplay
